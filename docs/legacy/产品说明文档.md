# AI-DevKit - 产品说明文档

## 📋 项目概述

AI-DevKit 是一个基于 React + TypeScript + Vite 的现代化前端项目模板，专为快速 AI 辅助开发而设计。该项目采用最佳实践架构，集成了完整的开发工具链，让开发者能够快速搭建高质量的前端应用。

### 🎯 项目目标

1. **让不会前端的人能通过 AI 快速完成前端项目搭建**
2. **给开发者提供一套一站式 AI 开发全栈平台**
3. **提供清晰的项目结构和最佳实践，便于 AI 理解和扩展**

### ✨ 核心特性

- 🏗️ **清晰的项目结构** - 基于功能模块划分 (Feature-based Structure)
- 🔧 **完整的开发工具链** - TypeScript、ESLint、Prettier、Jest、Husky
- 🎨 **现代化UI框架** - Tailwind CSS + Headless UI + Heroicons
- 🌐 **国际化支持** - i18next 多语言解决方案
- 🔐 **认证系统** - Supabase 原生集成
- 📱 **响应式设计** - 移动端友好的UI组件
- 🧪 **测试支持** - Jest + Testing Library
- 🚀 **快速部署** - Vercel 一键部署支持

## 🏗️ 项目架构

### 目录结构

```
src/
├── app/                    # 应用级配置和路由
├── features/              # 功能模块 (按业务划分)
│   ├── auth/             # 认证模块
│   ├── home/             # 首页模块
│   └── task/             # 任务模块
├── shared/               # 共享资源
│   ├── components/       # 通用组件
│   ├── hooks/           # 自定义Hooks
│   ├── services/        # API服务
│   ├── stores/          # 状态管理 (Zustand)
│   ├── types/           # TypeScript类型定义
│   ├── utils/           # 工具函数
│   ├── constants/       # 常量定义
│   └── i18n/            # 国际化配置
├── styles/              # 全局样式
├── mock/                # 模拟数据
└── examples/            # 示例代码
```

## 🎯 使用方法

### 1. 添加新功能模块

在 `src/features/` 目录下创建新的功能模块：

```typescript
// src/features/example/index.ts
export { default as ExamplePage } from './ExamplePage';
export { default as ExampleComponent } from './ExampleComponent';
export * from './types';
export * from './hooks';
export * from './stores';
```

### 2. 创建新组件

在 `src/shared/components/` 或功能模块内创建组件：

```typescript
// src/shared/components/Button/Button.tsx
import React from 'react';
import { cn } from '@/shared/utils/cn';

interface ButtonProps {
  variant?: 'primary' | 'secondary';
  size?: 'sm' | 'md' | 'lg';
  children: React.ReactNode;
  className?: string;
  onClick?: () => void;
}

export const Button: React.FC<ButtonProps> = ({
  variant = 'primary',
  size = 'md',
  children,
  className,
  onClick,
}) => {
  return (
    <button
      className={cn(
        'rounded-md font-medium transition-colors',
        {
          'bg-blue-600 text-white hover:bg-blue-700': variant === 'primary',
          'bg-gray-200 text-gray-900 hover:bg-gray-300': variant === 'secondary',
          'px-3 py-1 text-sm': size === 'sm',
          'px-4 py-2': size === 'md',
          'px-6 py-3 text-lg': size === 'lg',
        },
        className
      )}
      onClick={onClick}
    >
      {children}
    </button>
  );
};
```

### 3. 状态管理

使用 Zustand 创建状态管理：

```typescript
// src/features/example/stores/exampleStore.ts
import { create } from 'zustand';

interface ExampleState {
  data: any[];
  loading: boolean;
  fetchData: () => Promise<void>;
}

export const useExampleStore = create<ExampleState>((set) => ({
  data: [],
  loading: false,
  fetchData: async () => {
    set({ loading: true });
    try {
      // API调用逻辑
      const response = await fetch('/api/data');
      const data = await response.json();
      set({ data, loading: false });
    } catch (error) {
      set({ loading: false });
    }
  },
}));
```

### 4. API服务

在 `src/shared/services/` 中创建API服务：

```typescript
// src/shared/services/api.ts
import { supabase } from '@/shared/utils/supabase';

export const apiService = {
  async getData() {
    const { data, error } = await supabase
      .from('table_name')
      .select('*');
    
    if (error) throw error;
    return data;
  },
  
  async createData(payload: any) {
    const { data, error } = await supabase
      .from('table_name')
      .insert(payload)
      .select();
    
    if (error) throw error;
    return data;
  },
};
```

### 5. 路由配置

在 `src/app/routes.tsx` 中添加新路由：

```typescript
import { ExamplePage } from '@/features/example';

export const routes = [
  {
    path: '/example',
    element: <ExamplePage />,
  },
  // 其他路由...
];
```
