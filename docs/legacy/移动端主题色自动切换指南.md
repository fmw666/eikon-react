# 移动端主题色自动切换指南

## 概述

本指南介绍如何在 AI-DevKit 项目中实现移动端主题色的自动切换功能。当用户在移动端使用应用时，浏览器界面外的主题色会跟随应用主题自动切换，提供更好的用户体验。

## 功能特性

- ✅ 自动更新 `meta theme-color` 标签
- ✅ 动态更新 `site.webmanifest` 中的主题色
- ✅ 支持浅色/深色主题切换
- ✅ 自动适配系统主题偏好
- ✅ 支持 PWA 安装后的主题色显示
- ✅ 提供自定义主题色配置

## 实现原理

### 1. 主题色管理器 (`themeColorManager.ts`)

核心工具类，负责：
- 更新 HTML meta 标签的 `theme-color` 属性
- 动态生成并更新 webmanifest 文件
- 提供主题色配置管理

### 2. 主题色 Hook (`useThemeColor.ts`)

React Hook，提供：
- 自动监听主题变化
- 支持自定义主题色配置
- 返回当前主题色信息

### 3. 主题提供者集成

在 `ThemeProvider` 中集成主题色更新逻辑，确保主题切换时自动更新移动端主题色。

## 使用方法

### 基础使用

```tsx
import { useThemeColor } from '@/shared/hooks/useThemeColor';
import { useIsDarkMode, useSetTheme } from '@/app/providers/theme';

const MyComponent = () => {
  const isDarkMode = useIsDarkMode();
  const setTheme = useSetTheme();
  const { currentColors } = useThemeColor();

  const handleThemeToggle = () => {
    setTheme(isDarkMode ? 'light' : 'dark');
  };

  return (
    <div>
      <p>当前主题色: {currentColors.themeColor}</p>
      <button onClick={handleThemeToggle}>
        切换主题
      </button>
    </div>
  );
};
```

### 自定义主题色

```tsx
import { useThemeColor } from '@/shared/hooks/useThemeColor';

const MyComponent = () => {
  const { currentColors } = useThemeColor({
    lightThemeColor: '#10B981',      // 浅色主题色
    lightBackgroundColor: '#f9fafb', // 浅色背景色
    darkThemeColor: '#059669',       // 深色主题色
    darkBackgroundColor: '#1f2937',  // 深色背景色
  });

  return (
    <div>
      <p>自定义主题色: {currentColors.themeColor}</p>
    </div>
  );
};
```

### 直接使用工具函数

```tsx
import { updateThemeColor, setCustomThemeColors } from '@/shared/utils/themeColorManager';

// 更新主题色
updateThemeColor(true); // 深色模式
updateThemeColor(false); // 浅色模式

// 设置自定义主题色
setCustomThemeColors(
  '#10B981', // 浅色主题色
  '#f9fafb', // 浅色背景色
  '#059669', // 深色主题色
  '#1f2937'  // 深色背景色
);
```

### HTML 初始化

在 `index.html` 中已经集成了主题色初始化逻辑：

```html
<script>
  (function () {
    try {
      var stored = localStorage.getItem('theme');
      var resolved = stored ? (function (t) { try { return JSON.parse(t); } catch (e) { return t; } })(stored) : 'system';
      var prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
      var isDark = resolved === 'dark' || (resolved !== 'light' && prefersDark);
      var root = document.documentElement;
      root.classList.remove('light', 'dark');
      root.classList.add(isDark ? 'dark' : 'light');
      
      // 初始化移动端主题色
      var themeColors = {
        light: { themeColor: '#4F46E5', backgroundColor: '#ffffff' },
        dark: { themeColor: '#1f2937', backgroundColor: '#111827' }
      };
      var colors = isDark ? themeColors.dark : themeColors.light;
      
      // 更新 meta theme-color
      var metaThemeColor = document.querySelector('meta[name="theme-color"]');
      if (metaThemeColor) {
        metaThemeColor.setAttribute('content', colors.themeColor);
      }
    } catch (e) {}
  })();
</script>
```

## 注意事项

### 1. 性能考虑

- 主题色更新是轻量级操作，不会影响性能
- 使用 Blob URL 动态更新 webmanifest，避免文件系统操作

### 2. 兼容性

- 支持所有现代浏览器
- 在旧版浏览器中会优雅降级

### 3. 调试

如果主题色没有正确更新，请检查：

1. 确认 `meta[name="theme-color"]` 标签存在
2. 确认 `link[rel="manifest"]` 标签存在
3. 检查浏览器控制台是否有错误信息
4. 确认主题切换逻辑正常工作

### 4. 自定义配置

如需修改默认主题色，可以：

1. 直接修改 `themeColorManager.ts` 中的 `THEME_COLORS` 常量
2. 使用 `setCustomThemeColors` 函数动态设置
3. 通过 `useThemeColor` Hook 传入自定义配置
